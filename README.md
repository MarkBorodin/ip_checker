# INSTALL_APP


### Setup


clone repository:
```
git clone https://github.com/MarkBorodin/ip_checker.git
```

move to folder "ip_checker":
```
cd ip_checker
```


### run this in docker

run:

```
python manage.py migrate
backend python manage.py createsuperuser
python manage.py runserver
```

follow the link:
```
http://127.0.0.1/admin/
```


### Finish
